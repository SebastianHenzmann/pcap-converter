# pcap-converter

# Convert .pcap files into .hccapx files (2500 format) in PowerShell

# The script requires:

Hashcat: https://github.com/hashcat/hashcat

cap2hccapx: https://github.com/hashcat/hashcat-utils

PowerShell 5.1 or newer
